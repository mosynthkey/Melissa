Melissa

AI Audio Source Separation Feature-equipped music player for musical instrument practice for macOS / Windows
Free & Open source(LGPL-2.1)

  • GitHub
  • X
Scroll

Make your instrument practice more efficient and more enjoyable.
Melissa strongly supports your music instrument practice.

Support for repetitive practice

Efficiently supports repetitive practice, which is the key to improvement.

  • Playback speed can be changed without changing the pitch.
  • The repeat range can be easily set by dragging the waveform.
  • The set range can be saved and easily recalled.

Music Separation / Isolation Feature

AI based music separation feature makes it easier to hear the sound of each instrument.

  • It is possible to separate the sound into vocals/guitar/bass/drums/piano/other instruments, and to adjust the volume balance and solo playback.

Flexible keyboard shortcuts / MIDI controller operation

Keyboard shortcuts and MIDI controllers allow you to operate Melissa more easily and flexibly.

  • Almost all functions can be assigned to keyboard or MIDI controller operations.

File management function

Finding song files can become a hassle as the number of songs you practice increases. Melissa manages files on your behalf.

  • playlist function that allows you to register a set list of your live performance and play them in song order.
  • History list for easy retrieval of files that have been opened once
  • File browser for easily reading songs in the same folder

Metronome synchronized with the song

Supports practice at the exact tempo, in sync with the song.

  • When a song is loaded, the BPM is automatically detected.
  • By setting the accent position, the metronome can sound in sync with the song.
  • The metronome follows the playback speed of the song.

Simple and intuitive UI

Consisting of a single screen, all functions can be accessed with a few clicks.
