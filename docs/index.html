楽器練習をもっと効率的に、より楽しく。
あなたの練習を強力にサポートします。

反復練習のサポート

上達の鍵となる反復練習を効率的に行うことができます。

  • 音程を変えずに再生速度を変更することができます。
  • リピートする範囲は波形をドラッグすることで簡単に設定できます。
  • 設定した範囲は保存することができ、簡単に呼び戻すことができます。

曲と同期するメトロノーム

曲に合わせた、正確なテンポでの練習をサポートします。

  • 曲を読み込むと、自動的にBPMを検出します。
  • アクセント位置を設定することで、曲に同期してメトロノームを鳴らすことができます。
  • メトロノームは曲の再生速度に追従します。

ファイル管理機能

練習曲が増えると面倒になる曲のファイル探し。あなたに代わってファイル管理します。

  • ライブのセットリストの曲を入れておくと便利なプレイリスト機能
  • 一度開いたファイルを簡単に探し出せるHistoryリスト
  • 同じフォルダにある曲を簡単に読み出せる>ファイルブラウザ

シンプルで直感的なUI

1画面で構成されており、すべての機能に数クリックでアクセスできます。
