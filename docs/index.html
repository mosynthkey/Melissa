楽器練習をもっと効率的に、より楽しく。
あなたの練習を強力にサポートします。

反復練習のサポート

上達の鍵となる反復練習を効率的に行うことができます。

  • 音程を変えずに再生速度を変更することができます。
  • リピートする範囲は波形をドラッグすることで簡単に設定できます。
  • 設定した範囲は保存することができ、簡単に呼び戻すことができます。

楽器ごとの音声分離機能

機械学習による音声分離機能により、楽器ごとの音をより聴き取りやすくすることができます。

  • ボーカル/ギター/ベース/ドラム/ピアノ/その他の楽器に分離することができ、ソロ再生や音量バランスの調整も可能です。

柔軟なキーボードショートカット / MIDIコントローラーでの操作

キーボードショートカットやMIDIコントローラーで、より簡単かつ柔軟にMelissaを操作できます。

  • ほとんどすべての機能にキーボードやMIDIコントローラーでの操作を割り当てることができます。

ファイル管理機能

練習曲が増えると面倒になる曲のファイル探し。あなたに代わってファイル管理します。

  • ライブのセットリストの曲を入れておくと便利なプレイリスト機能
  • 一度開いたファイルを簡単に探し出せるHistoryリスト
  • 同じフォルダにある曲を簡単に読み出せるファイルブラウザ

曲と同期するメトロノーム

曲に合わせた、正確なテンポでの練習をサポートします。

  • 曲を読み込むと、自動的にBPMを検出します。
  • アクセント位置を設定することで、曲に同期してメトロノームを鳴らすことができます。
  • メトロノームは曲の再生速度に追従します。

シンプルで直感的なUI

1画面で構成されており、すべての機能に数クリックでアクセスできます。
